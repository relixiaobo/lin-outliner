Tag Layout Pattern

Spec: components.md Pattern: gapped relative layout Token: tag palette

Applied tags render inline after node text. The normal state keeps the hash and label inside one measured pill. On hover or keyboard focus, the same measured box swaps to a remove target plus a label-only pill, so row height and surrounding text do not move.
