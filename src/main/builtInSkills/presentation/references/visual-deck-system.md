# Visual Deck System

## Temperaments

Choose one visual temperament and carry it through the deck.

### Editorial Narrative

Use for talks, essays, opinions, culture, launches, and storytelling.

- strong section rhythm
- display headlines
- full-bleed or half-bleed imagery
- large quotes
- generous whitespace
- cinematic transitions

### Grid Analytical

Use for strategy, product, facts, frameworks, research, and business cases.

- strict alignment
- restrained palette
- one accent color
- sharp rectangles and hairline rules
- strong type-scale contrast
- minimal decoration

## Layout Variety

Use a mix when appropriate:

- cover
- agenda or thesis
- section divider
- two-column explanation
- image-led point
- comparison
- process or timeline
- stat page
- chart page
- quote or callout
- grid of examples
- closing action

Avoid repeating title-plus-bullets unless the user requested a plain outline.

## Slide Composition

- Make the headline a claim, not a label.
- Keep body text speaker-readable.
- Avoid tiny dense tables on projected slides.
- Give screenshots and diagrams enough room.
- Use consistent margins and alignment.
- Keep text inside safe areas for covers and cropped exports.

## Asset Rules

- Prefer real product, place, person, chart, diagram, or generated bitmap imagery over decorative abstraction.
- Use stable image ratios: 16:9, 21:9, 16:10, 4:3, 3:2, 1:1, 3:4.
- Crop intentionally; keep subjects in safe areas.
- Use contain-fit for UI screenshots when readability matters.
