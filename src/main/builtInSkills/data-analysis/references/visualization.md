# Visualization

## Chart Choice

- Trend over time: line chart.
- Category comparison: bar chart.
- Part-to-whole: stacked bar only when components matter; avoid pie charts unless the user asks.
- Distribution: histogram, box plot, or dot plot.
- Relationship: scatter plot with clear axes and units.
- Table: use when exact values matter more than shape.

## Visual QA

- Use clear titles that state the finding.
- Label axes, units, and time ranges.
- Keep scales honest and readable.
- Avoid dense legends and tiny labels.
- Do not decorate charts with unrelated imagery.
- Verify that chart data matches the transformed dataset.

## Delivery

- Include the chart source data or transformation notes.
- State filters and denominators near the chart.
- Prefer accessible colors and direct labels.
